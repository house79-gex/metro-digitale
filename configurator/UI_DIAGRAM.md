## Responsive Behavior

```
Window Resize:
- Toolbox: Fixed width (left dock)
- Properties: Fixed width (right dock)
- Canvas: Center, maintains aspect ratio
- Editors: Bottom dock, resizable height

Canvas Zoom:
- Transform applied to QGraphicsView
- Maintains coordinate system
- Scroll bars appear when needed

Element Movement:
- Constrained to display bounds (0-800, 0-480)
- Snap to grid when enabled
- Real-time coordinate updates in footer
```
